Question: What are the titles of publications authored by a specific author?
Query:

cypher
Copy code
MATCH (a:Author)-[:AUTHORED]->(p:Publication) 
WHERE a.name = 'Author Name' 
RETURN p.title

Question: What are the top 10 cited publications?
Query:

cypher
Copy code

RETURN p.title, count(citing) AS citation_count

LIMIT 10

cypher
Copy code
